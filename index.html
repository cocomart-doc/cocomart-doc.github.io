建立或更新採購商物料

建立採購商物料，若已經存在，則進行更新

URL

POST /partner-api/integration/v1/purchaser/purchaserItem

Request

輸入參數
欄位 必填 型別 長度 說明 備註
sourceItemId Y string 30 來源系統的產品唯一識別編號。 若CocoMart系統未存在該識別編號，則視為新增；若已存在，則進行資料更新。
name Y string 150 產品名稱
refNumber N string 100 供應商管理料號
spec N string 30 產品規格
category N string 10 產品類別
description N string 300 產品敘述
unit N string 10 單位
Request 範例
{
  "sourceProductId": 10001,
  "name": "高麗菜",
  "refNumber": "CCM001",
  "spec": "5顆/箱",
  "category ": "蔬菜",
  "description": "高山高麗菜(門市用)",
  "unit": "顆"
}

Response

輸出參數

無

輸出範例

請求成功 HTTP Code 200

{
  "data": {}
}
